# 🚀 DevOps Theme Portfolio Website

A stunning, highly-interactive, and ultra-premium single-page portfolio website designed for DevOps & Site Reliability Engineers. The theme replicates a professional Cloud Cluster Management Console featuring live dashboards, telemetry graphs, and automation widgets.

## 🛠️ Tech Stack & Architecture
- **Structure**: Semantic HTML5 markup
- **Styling**: Modern, responsive vanilla CSS (cyberpunk dark-theme, glassmorphism, glowing replica state controllers, custom keyframe animation matrices)
- **Logic**: Vanilla JavaScript ES6 (Terminal CLI emulator, SVG pipeline engine, Canvas 2D live telemetry streams, cluster pod orchestrator, mock REST API handler)
- **No Dependencies Required**: Runs instantly in any modern web browser without a compiler, build tool, or dev server.

---

## 🎮 Core Features

1. **Interactive Terminal CLI**:
   - A fully-draggable, resizable retro-modern console.
   - Command parser with auto-scrolling, history buffer (Up/Down arrow keys), and custom styling theme triggers (`theme classic`, `theme neon`, `theme matrix`).
   - Run `help` in the shell to see the full operations manual!

2. **CI/CD Pipeline Simulator**:
   - Visual step-by-step progress tracking for standard pipelines (Git webhook -> Lint audit -> Docker build -> Unit tests -> ECR registry -> K8s deployment).
   - Dynamic step visual states (active rotation, success ticks, glowing indicators).
   - Real-time build stdout log tailing directly to the Pipeline Console.

3. **Telemetry & Cluster Monitoring Dashboard**:
   - High-performance, real-time updated Canvas graphs tracking virtual CPU & Memory drift metrics.
   - Interactive Kubernetes Namespace cluster grid showing active pods.
   - Click pods to view isolated container telemetry details or **Trigger Pod Restarts** to watch standard container evictions and replicas scheduling in real-time.
   - Live tailing System Logs feed mapping web requests and controller actions.

4. **Infrastructure-as-Code Profile**:
   - View my bio, technologies, and experience declared as DevOps configurations: `Dockerfile`, `deployment.yaml`, and `main.tf` specs.
   - Features mock syntax highlighting and a double-click "Copy Config" clipboard functionality.

5. **Contact REST API Gateway Client**:
   - Submit messages through a custom REST API Client interface (`POST /api/v1/contact`).
   - Returns a structured, syntax-highlighted JSON response (`201 CREATED`) reflecting gateway headers, ingress metrics, and payloads.

## ⚡ How to Run

Since the application is built entirely on native web standards (HTML5/CSS3/ES6 JS), there are no complex installation instructions.

### Option 1: Direct Execution (Easiest)
1. Navigate to the project folder:
   `C:\Users\Benaka\.gemini\antigravity\scratch\devops-portfolio\`
2. Double-click the [index.html](file:///C:/Users/Benaka/.gemini/antigravity/scratch/devops-portfolio/index.html) file to open it in Google Chrome, Microsoft Edge, Mozilla Firefox, or Safari!

### Option 2: Live Server (For Developers)
If you have a local server extension (like VS Code Live Server, Python simple HTTP server, or Node package `serve`), you can run it inside the directory:
```bash
# Python 3
python -m http.server 8000

# Node.js (if installed)
npx serve
```
Then navigate to `http://localhost:8000`.
